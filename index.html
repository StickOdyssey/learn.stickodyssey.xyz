Discovering HTML and Tags

What is HTML?

Html is a computer markup language that structures all the webpages on the internet. HTML stands for Hypertext Markup Language.

HTML Tags

All tags start with <, followed by a keyword, and end with >. Tags like <button> are opening tags, to finish creating a button element we need text, and a closing tag, we can make and a closing tag by adding a / to another tag, and then write what we want in between them.

Text and Headings

We can create headings in HTML to emphasize certain texts. There are 6 sizes of headings in HTML, from h1 to h6.

In HTML, the keywords of some tags are obvious, but there are other tags that use a shortened form. For example, we use the <p> tag for a paragraph. Coding a paragraph is similar to coding a button. You need an opening tag, followed by your text, and a closing tag.

Web Page Structuring

An element is what we call the opening tag, closing tag, and code in between.

To create a webpage, you’ll need an opening and closing HTML tag.

An element is what we call the opening tag, closing tag, and code in between.
